# Partner Integration

{% hint style="info" %}
The partner integration program is under development. Affiliate API and partner tools are planned for Q3 2026.
{% endhint %}

## What partners get

7.Exchange is building an integration layer that allows wallets, exchanges, dApps, and platforms to embed cross-chain swap routing directly into their products.

### Routing as a service

Partners integrate the 7.Exchange API or SDK and instantly gain access to the full routing engine — all supported chains, bridges, liquidity sources, and quote optimization — without building or maintaining any of that infrastructure.

### Revenue sharing

Partners earn a share of routing fees on every swap executed through their integration. Revenue is tracked via the affiliate API and attributed to the partner's integration key.

### No custody

The integration is non-custodial. Users interact with the partner's interface, and swaps are executed through the same trust-minimized pipeline as the main 7.Exchange app. Partners never touch user funds.

## Integration options

| Option | Description |
|---|---|
| **API** | Full control over the UX. Query quotes, build transactions, and track status via REST endpoints. |
| **SDK** | TypeScript library that wraps the API with helpers for common integration patterns. |
| **Widget** | An embeddable swap interface that can be dropped into any website or dApp with minimal code. |
