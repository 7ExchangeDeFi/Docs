# Developers

{% hint style="info" %}
The 7.Exchange developer platform is under development. API, SDK, and partner integration tools are planned for Q3 2026. Follow our updates on [X](https://x.com/7exchangeDeFi) and [Discord](https://discord.gg/your-server) for launch announcements.
{% endhint %}

7.Exchange is building a developer platform that will allow wallets, dApps, trading desks, and other applications to integrate cross-chain swap routing directly.

## What's coming

### API

A public REST API with authenticated endpoints for:

- Quote retrieval and route computation
- Transaction construction
- Transaction status tracking
- Supported chains, tokens, and providers

### SDK

A TypeScript SDK that wraps the API for easier integration in frontend and backend environments. Designed for minimal setup and full control over the swap flow.

### Partner integration

An affiliate API enabling partners to embed 7.Exchange routing with revenue-share tracking. Partners will earn fees on volume routed through their integration.

### Documentation

Full developer documentation with endpoint references, code examples, integration guides, and a sandbox environment for testing.

## Interested in early access?

If you're building a wallet, dApp, or trading product and want to integrate 7.Exchange routing before the public launch, reach out at [contact@7.exchange](mailto:contact@7.exchange).
